How to resize an image for a Discord emoji

  1. Click the upload box above (or drag your image onto it).
  2. The tool instantly resizes it to 128×128 pixels — the size Discord uses for emojis.
  3. It automatically compresses the file to stay under Discord's 256KB upload limit.
  4. Click Download Emoji and upload the file to your Discord server.

What size are Discord emojis?

Discord displays custom emojis at 128×128 pixels. When you upload an emoji, Discord requires the file to be 256KB or smaller. If your image is too big or the wrong shape, Discord may reject it or squish it. This tool fixes both problems in one step.
