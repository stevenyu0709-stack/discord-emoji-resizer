How to make a Discord sticker

  1. Click the upload box above (or drag your image onto it).
  2. The tool instantly resizes it to 320×320 pixels — the size Discord uses for stickers.
  3. It automatically compresses the file to stay under Discord's 512KB sticker limit.
  4. Click Download Sticker and upload it to your Discord server settings.

What size are Discord stickers?

Discord custom stickers must be 320×320 pixels and 512KB or smaller. PNG with transparency works best. This tool fixes both the size and file weight in one step.
